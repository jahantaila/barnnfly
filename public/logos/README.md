# Bark & Fly — Logo Concepts

Drop your logo concept images into this folder named `concept-1.png` through `concept-6.png`.

The survey's "Logo vote" step will automatically display them at `/logos/concept-N.png`.

Recommended specs:
- Format: PNG with transparent background (or JPG)
- Size: ~1200×900 (4:3 aspect)
- Max file size: under 400 KB each

If a concept image is missing, the card shows a placeholder with the expected path.
